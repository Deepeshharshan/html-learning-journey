Overview

This repository is a personal learning log created while studying HTML5 as the first step in my journey toward becoming a Full Stack Web Developer. It documents my progress in a structured, chapter-by-chapter format — starting from the very basics of document structure and moving toward more advanced topics like tables, forms, and semantic HTML.

The goal of this repository is not just to store code, but to build a genuine understanding of why HTML works the way it does. Each topic is broken down into its own folder containing:


Practice files — small, focused .html files demonstrating a single concept at a time
Comments and notes — inline explanations written in my own words to reinforce learning
Incremental complexity — later topics build on concepts introduced earlier


This repository is part of my broader Full Stack Web Development roadmap, which will eventually expand to include CSS, JavaScript, and backend technologies. HTML is the foundation of that roadmap — every element, tag, and structure learned here will directly support the styling and interactivity added in later stages.

Whether you're a fellow beginner following a similar path, a mentor reviewing my progress, or a recruiter looking at my learning process, this repository is meant to give a clear, transparent view of how I approach learning a new technology: practice first, document consistently, and build in public.
